Explicação dos slides 33 e 34

Os slides apresentam a árvore DOM gerada a partir de um documento HTML específico. Nessa estrutura, cada elemento da marcação (por exemplo, h1, input ou p) torna-se um nó-objeto que herda propriedades e métodos de uma hierarquia de interfaces (EventTarget -> Node -> Element -> HTMLElement, e assim por diante). Além desses nós de elemento, a árvore inclui nós de texto, comentários para que todo o conteúdo — inclusive o texto entre as tags — seja representado e possa ser manipulado via JavaScript.
